--- title: Opzetten van een eigen website ---

Week 4716/11/2015 - 20/11/2015

Ik heb besloten om mijn stageverslag in een vorm van een website te maken. Deze website wil ik dan ook helemaal vanaf de rond zelf bouwen. Door deze vorm kan je zien wat ik geleerd heb van Html, Css en Java Script. De website opzetten is via www.github.com gebeurt, al een hele taak. Daarna moet er handmatig de navigatie en de verschillende pagina's, besturingssystemen opgezet worden. Het is een heel proces wat je moet door gaan voor dat je pas een aantal basis pagina's heb.

Gewerkt aan de volgende doelen uit het stagewerkplan:
- Verdiepen in Css
- Verdiepen in Html
- Verdiepen in Java script

Leerervaringen
Mijn leerervaringen van deze week: Het bekijken van verschillende besturingssystemen. Het aanpassen van een al geschreven code. In al dit gaat al zo veel tijd in zitten.
