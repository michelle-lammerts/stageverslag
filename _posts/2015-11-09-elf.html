--- title: Bugs ---

Week 4609/11/2015 - 13/11/2015

Deze week heb ik een stagegesprek met mijn stagebegeleider Gerwin Veenstra en stage coördinator Maaike Hamelynck gehad. Na mijn idee ging de beoordeling goed, er waren niet veel op of aanmerkingen.
Dde rest van de weekben ik bezig geweest met het debuggen van LUMC en LUMC jaargesprekken. Er zatten grote fout in het inlogscherm als ze de site via IE opende. Het na deel van IE is dat hij niet up-to-date met verschillende codes van Html, Css en Java script is. We konden niet vinden waar de fout zat en hoe hij opgelost moest worden. Ik ben er achter gekomen dat Html en Css veel met oplossen van fouten en problemen te maken heeft. Als er al één code fout staat snapt de computer al niet meer wat je bedoeld.

Gewerkt aan de volgende doelen uit het stagewerkplan:
- Verdiepen in Css
- Verdiepen in Java script

Leerervaringen
Mijn leerervaringen van deze week: Hoe irrant IE wel niet is. Naast alle goede internet applicaties gebruiken veel bedrijven nog IE. Ze lopen zo veel mis en er gaat zo veel kapot.
