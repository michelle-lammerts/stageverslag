CAPP

Week 3814/09/2015 - 18/09/2015

Deze week heb ik mij verdiept in het oplossen van problemen in een Html en Css. Ik ben er achter gekomen dat je niet zomaar elk effect op een website met Html en Css kan implementeren. Vaak heb je hier Java Script voor nodig, daarom heb ik een start gemaakt in het leren van Java script via www.codeacademie.com.

Ook moest ik mij deze week verdiepen in CAPP dit is een software wat Defacto verkoopt aan bedrijven (zie het bedrijfsprofiel). Elk bedrijf heeft een andere uitstraling nodig, je veranderd hier bij de Css, Devexpress en sitemaps. Dit worden stylsheets genoemd.

Participatie

Week 37 07/09/2015 - 07/09/2015

Net zoals vorige week stond deze week in het teken van het leren van Html en Css. Ik leer deze scripts door middel van het maken van een eigen website. Natuurlijk vallen mij gelijk allerlei dingen op. Ik kan veel minder designen met Css dan verwacht. Er zitten zo veel haken en ogen aan. Dit is echt een heel ander vakgebied in design. Verder heb ik mij ook meer moeten verdiepen in de Command line. Alles wat je maakt, downloaden en doet kan je beter via de Command line doen. Je gebruikt dit programma voornamelijk als je bezig bent op Github, zo kan je makkelijk je scripts pushen, klonen en pullen van deze site.

Voor de rest ben ik een beetje gaan participeren in het bedrijf, zoals mee doen met bepaalde activiteiten, zoals samen lunchen en tafel voetballen.
