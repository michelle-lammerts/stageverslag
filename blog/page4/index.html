medicijn venijn

Week 4028/09/2015 - 03/10/2015

Het ontwerpen van een medicatiedoosje bleek iets meer werk te zijn dan een paar uur. Door slordige communicatie en het steeds veranderen van ideeën, heb ik hier meer tijd in moeten steken dan eigenlijk zou moeten.
Ook moest ik beginnen met het ontwerpen van een bijsluiter die in het medicatiedoosje komt. Uit eindelijk na veel keuringen kon ik de bijsluiters en de stickers die op de doosjes kwamen, laten drukken. Dit waren 500 doosjes en 1000 stickers. Naast het beplakken van de doosje moesten de doosjes ook nog gevouwen worden.

Daar naast had Defacto een nieuw Duits logo nodig. Hier heb ik ontwerpen voor aangeleverd.

Gewerkt aan de volgende doelen uit het stagewerkplan:
- Ontwerpen van promotie materiaal voor het bedrijf

Leerervaringen
Mijn leerervaringen van deze week: Mijn leerervaring deze week was toch wel het omgaan met broede communicatie. Het was stresvol en frustrerend, soms moest ik mij inhouden om mijn gedachten niet uit te spreken. Was er een idee goedgekeurd en het ontwerp zo goed als af, kwamen ze weer op het laatste moment met een ander idee. Dus leerervarigen van deze week waren: omgaan met stres, frustratie en het niet uitspreken van mijn gedachtens.

Participatie

Week 37 07/09/2015 - 07/09/2015

Net zoals vorige week stond deze week in het teken van het leren van Html en Css. Ik leer deze scripts door middel van het maken van een eigen website. Natuurlijk vallen mij gelijk allerlei dingen op. Ik kan veel minder designen met Css dan verwacht. Er zitten veel haken en ogen aan. Dit is echt een heel ander vakgebied in design dan verwacht.
Verder heb ik mij meer moeten verdiepen in de Command line. Alles wat je maakt, downloaden en doet kan je beter via de Command line doen. Je gebruikt dit programma voornamelijk als je bezig bent op Github, zo kan je makkelijk je scripts pushen, klonen en pullen.
Voor de rest ben ik een beetje gaan participeren in het bedrijf, zoals mee doen met bepaalde activiteiten, zoals samen lunchen en tafel voetballen.

Gewerkt aan de volgende doelen uit het stagewerkplan:
- Verdiepen in Css
- Verdiepen in html

Leerervaringen
Mijn leerervaringen van deze week: Deze week heb ik mij al veel meer kunnen verdiepen in Html en Css. Door het creëren van een eigen website loop je snel tegen bepaalde design problemen aan. Het leren wat wel en wat niet kan in Html en Css is denk ik het belangrijkste wat ik komende periode ga leren.
De command line is nummer twee van mijn leerervaringen deze week. Wat ik geleerd van de command line heb is dit: Wanneer een shell of computerprogramma de gebruiker een command line interface (ook wel opdrachtregelinterface, commandoregel, terminal of command line interface) aanbiedt, wil dat zeggen dat de gebruiker het aanspreekt aan de hand van regels tekst die een of meerdere opdrachten bevatten. Nadat de opdracht is uitgevoerd krijgt de gebruiker weer de kans om de shell of het programma aan te spreken door op de opdrachtregel een nieuwe opdracht op te geven.
