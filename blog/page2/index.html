Upgraden

Week 4823/11/2015 - 27/11/2015

Deze week heb ik weer promotie materiaal moeten maken voor een Duitse medicatie beurs, alleen dit keer was het een banner. De banner moest 1 meter bij 0,50 meter worden. Natuurlijk moest de styl van defacto terug komen in de banner. De rest van de week ben ik bezig geweest om NFI naar een nieuwere versie van CAPP te zetten, een hele taak.

Gewerkt aan de volgende doelen uit het stagewerkplan:
- Verdiepen in Css
- Ontwerpen van promotie materiaal voor het bedrijf

Leerervaringen
Mijn leerervaringen van deze week: Deze week heb ik veel meer geleerd hoe de css van de stylsheets in elkaar zitten. Ze zitten vrij lastig in elkaar, je kan de omgeving met één verandering al helemaal stuk maken.

Opzetten van een eigen website

Week 4716/11/2015 - 20/11/2015

Ik heb besloten om mijn stageverslag in een vorm van een website te maken. Deze website wil ik dan ook helemaal vanaf de rond zelf bouwen. Door deze vorm kan je zien wat ik geleerd heb van Html, Css en Java Script. De website opzetten is via www.github.com gebeurt, al een hele taak. Daarna moet er handmatig de navigatie en de verschillende pagina's, besturingssystemen opgezet worden. Het is een heel proces wat je moet door gaan voor dat je pas een aantal basis pagina's heb.

Gewerkt aan de volgende doelen uit het stagewerkplan:
- Verdiepen in Css
- Verdiepen in Html
- Verdiepen in Java script

Leerervaringen
Mijn leerervaringen van deze week: Het bekijken van verschillende besturingssystemen. Het aanpassen van een al geschreven code. In al dit gaat al zo veel tijd in zitten.

Bugs

Week 4609/11/2015 - 13/11/2015

Deze week heb ik een stagegesprek met mijn stagebegeleider Gerwin Veenstra en stage coördinator Maaike Hamelynck gehad. Na mijn idee ging de beoordeling goed, er waren niet veel op of aanmerkingen.
Dde rest van de weekben ik bezig geweest met het debuggen van LUMC en LUMC jaargesprekken. Er zatten grote fout in het inlogscherm als ze de site via IE opende. Het na deel van IE is dat hij niet up-to-date met verschillende codes van Html, Css en Java script is. We konden niet vinden waar de fout zat en hoe hij opgelost moest worden. Ik ben er achter gekomen dat Html en Css veel met oplossen van fouten en problemen te maken heeft. Als er al één code fout staat snapt de computer al niet meer wat je bedoeld.

Gewerkt aan de volgende doelen uit het stagewerkplan:
- Verdiepen in Css
- Verdiepen in Java script

Leerervaringen
Mijn leerervaringen van deze week: Hoe irrant IE wel niet is. Naast alle goede internet applicaties gebruiken veel bedrijven nog IE. Ze lopen zo veel mis en er gaat zo veel kapot.

Over smaak valt niet te twisten

Week 4502/11/2015 - 06/11/2015

Deze week moest ik de stylsheets van Trajectum, LUMC, Raamwerk en NFI aanpassen. Het is soms echt moeilijk om naar de klanten te luisteren, je weet van te voren dat het niet mooi gaat worden en toch willen ze het. Dan maak je er wat van, maar het is nog steeds niet goed of mooi en zei vinden het prachtig. Het is soms heel moeilijk om niet gillend gek te worden. Bijvoorbeeld een bedrijf zoals Tui, je zou zeggen een groot bedrijf die wel een goed design wil hebben. Maar, nee helaas. Niet alleen moet ik rare kleur stellingen bij elkaar gebruiken, ze sturen mij ook pixelige bestanden. Het bedrijf Raamwerk stuurde mij zelfs een logo via Paint.

Gewerkt aan de volgende doelen uit het stagewerkplan:
- Verdiepen in Css

Leerervaringen
Mijn leerervaringen van deze week: Dit is wat ik nog graag wil leren, een klant duidelijk uitleggen of een design idee die ze hebben wel of niet kan.
