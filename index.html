Grafisch design vs functionaliteit

“Eenvoud is de ultieme verfijning". (Leonardo Davinci)

Voor dat ik ga vertellen over wat ik in deze stageperiode heb gedaan, wil ik graag iets kwijt over het thema waar ik mijn hele stage aan gewijd heb, “Functionaliteit”. Waarom is het bereiken van de eenvoud zo lastig in webdesign? Juist de simpele websites zijn makkelijk te gebruiken, te begrijpen en leuk om naar te kijken. Waarom wil ik onbewust vaak iets extra's toevoegen aan mijn design's terwijl het totaal niet nodig is?

Door Css, Html en Java Script te leren, kwam ik er achter dat heel veel van mijn ideeën niet bruikbaar zijn in de web/app design. Om mijn stage hier aan te wijden heb ik geleerd wat deze programma's allemaal niet en wel kunnen. Daarom is het ook belangrijk dat Minerva meer les geeft rond om deze sector. Niet alleen zoekt deze sector designers met Html, Css en Java script ervaringen, maar het is ook de toekomst waar grafisch design heen gaat.
